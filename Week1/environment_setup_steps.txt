1. Create a virtual environmentconda create --name MSDS694 python=3.6 -y2. Activating the environmentsource activate MSDS6943. Determining the current environmentconda info --envs4. List environment conda env list5. Deactivate the environmentsource deactivate6. Activate againsource activate MSDS6947. Export the environment file to share and reproduce the environments including all the packages with corresponding versions.conda install numpyconda env export > MSDS694_environment.yml8. Remove the environmentsource deactivateconda remove --name MSDS694 --all9.Create or update an environment using .yml.conda env create -f MSDS694_environment.yml -n MSDS694
